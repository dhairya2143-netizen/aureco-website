# Aureco Website

Premium packaging solutions for fashion and jewelry brands.

## 🚀 Quick Deploy to Vercel + Supabase

**Cost: $10-15/year (just domain!)**

### Prerequisites:
- Domain (e.g., aureco.com)
- Supabase account (free) - https://supabase.com
- Vercel account (free) - https://vercel.com

### Deploy in 3 Steps:

1. **Set up Supabase:**
   - Create project at https://supabase.com
   - Run SQL from `SUPABASE_SETUP.md`
   - Get API credentials

2. **Deploy to Vercel:**
   - Import project to https://vercel.com
   - Add environment variables
   - Deploy!

3. **Connect domain:**
   - Add domain in Vercel
   - Update DNS records
   - SSL auto-provisioned ✅

**Full Guide:** See `DEPLOYMENT_GUIDE.md` for step-by-step instructions
